# FounderOS

FounderOS is the unified operating system for all BearifiedCo projects.

It integrates:

- Slack (event layer)
- Notion (knowledge + roadmap)
- Linear (execution)
- GitHub (code + PRs)
- Gmail (CRM + inbound leads)
- Cursor, Codex, Claude Code, ChatGPT (AI executive team)

This repo stores:

- Agent instructions
- Automation rules
- Architecture diagrams
- Roadmap management
- Product specifications
- Observability plans

## Structure

- /agents — AI agent definitions and rules
- /notion — Notion workspace architecture
- /linear — Linear workflow specs
- /slack — Slack automation rules
- /gmail — CRM automation
- /alpha-builder — MVP specs for Construction AI app
- /observability — Datadog integration plans

This repo powers all Bearified products:

- BEARO (payments app)
- AlphaBuilder (construction estimating)
- Primape (prediction gaming)
- Chimpanion (wallet assistant)
- BEARCO/APES token ecosystem
