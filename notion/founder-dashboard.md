# Founder Dashboard (Notion Blueprint)

This dashboard pulls together:

## 1. Linked Databases

- Roadmaps DB (group by Product + Status)
- CRM (filtered by Active)
- Content Machine (Kanban: Idea → Posted)
- Contributors list
- Weekly Objectives (auto from Linear)

## 2. Today's Focus

List of 3–5 Linear P1 tasks.

## 3. Product Spaces

Buttons linking to:

- BEARO Hub
- AlphaBuilder Hub
- Primape Hub
- Chimpanion Hub

## 4. CEO Notes

A daily founder journal section.

## 5. Stream Integration

Optional view for "Now Building" based on Linear tasks in progress.
